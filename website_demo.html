Phase 1 Website Integration

One widget, three embed patterns.

This page demonstrates the same chatbot running as a floating launcher, popup panel, and mobile-friendly inline embed.

Open this page locally and you can test the exact integration patterns you would ship on a website.

Local demo URLs

Widget UI: https://ai-chatbot-mvp-v2.netlify.app/widget.html

Popup launcher appears in the bottom-right corner of this page automatically.

Bottom-right launcher

Add one script tag and the launcher opens a popup chat panel from the bottom-right corner.

<script src="widget-loader.js" data-mode="popup"></script>

Popup chat panel

The popup uses the same widget page in an iframe and automatically becomes full-screen on small screens.

data-backend-http="http://127.0.0.1:5001"

Mobile-friendly embed

For content pages, mount an inline iframe that grows to full width and uses full-height on mobile.

data-mode="inline" data-inline-target="#inline-widget-target"
